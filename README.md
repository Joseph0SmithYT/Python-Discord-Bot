# Quortle


A simple Discord bot dedicated to storing your quotes! 

# Commands
## Consumer Commands
 - `add_quote`
   - Adds a quote to the quote book.
 - `remove_quote`
   - Removes a quote from the quote book.
 - `get_quotes`
   - Fetches the quotes in a easy-to-read format
